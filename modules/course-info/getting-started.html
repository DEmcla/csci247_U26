Getting Started: Before Week 1

Welcome to Confident Uncertainty

This page helps you prepare for the course before Week 1 officially begins. Completing these tasks now means you can focus on learning—not logistics—when the course starts.

Time needed: 30-45 minutes

A Note About the "CSCI" in the Course Number

Yes, this course has a Computer Science prefix. No, you don't need to know anything about computer science to succeed.

This course requires:

  • No coding
  • No math beyond basic arithmetic
  • No prior technical knowledge

What it does require: curiosity, critical thinking, and willingness to ask questions. If you can use a smartphone and write a coherent paragraph, you have the prerequisites.

The course is designed for students from all majors. Some of the best evaluators are students who ask "obvious" questions that technical people overlook.

Pre-Course Checklist

  • Access to required AI tools (free accounts)
  • Microsoft Word installed (desktop version)
  • Learning Portfolio document created
  • Read the course overview

Task 1: Set Up AI Tool Access

Create free accounts for these AI tools (if you don't already have them):

  1. ChatGPT — chat.openai.com (free tier is sufficient)
  2. Claude — claude.ai (free tier is sufficient)
  3. Google Gemini — gemini.google.com (requires Google account)

You'll use all three during the course to compare how different AI tools handle the same questions.

Task 2: Verify Microsoft Word Access

Your Learning Portfolio must be created in Microsoft Word (desktop version). Verify you have access:

  • MCLA students: Word is available free through Office 365. Go to portal.office.com and sign in with your MCLA credentials.
  • Important: You need the desktop application, not the web version. The web version doesn't support all the formatting features required for the portfolio.

If you have trouble accessing Word, contact IT Help Desk before the course starts.

Task 3: Create Your Learning Portfolio Document

Create your portfolio document now so you're ready to add content in Week 1.

Step-by-Step Instructions

  1. Open Word and create a new blank document
  2. Save the file as "CSCI247_Portfolio_YourLastName.docx"
  3. Create a cover page with:
    • Course title: CSCI 247: Confident Uncertainty
    • Your name
    • Date (update this when you submit)
  4. Insert a Table of Contents (References → Table of Contents → Automatic Table 1)
  5. Create section headings using Heading 1 style for these six sections:
    • 1. Content Knowledge
    • 2. Problem Solving
    • 3. Communication
    • 4. Graphs & Visualization
    • 5. Scientific Practices
    • 6. Source Evaluation
  6. Add a final section: "Synthesis Reflection" (Heading 1)
  7. Update your Table of Contents (right-click → Update Field → Update entire table)
  8. Save your document

Why Set This Up Now?

Creating the document structure now means you won't be learning Word skills under deadline pressure later. When you encounter a good artifact during Week 1, you can immediately add it to the right section.

See the Learning Portfolio Guide for complete details on what goes in each section.

Task 4: Familiarize Yourself with the Course

Briefly review these documents to understand what you're getting into:

  • Course Requirements — What you need, what's expected
  • The Evaluation Framework — Skim to see the six components you'll learn (don't worry about memorizing them yet)

You don't need to read these in detail. Just get a sense of the course structure.

You're Ready

Once you've completed these tasks, you're prepared for Week 1. The first week will focus on understanding what AI actually is and how it works—no prior knowledge required.

If you have questions before the course starts, post them in the Q&A discussion forum or email the instructor.
