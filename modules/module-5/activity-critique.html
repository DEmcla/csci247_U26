Critique the Explanation

Purpose: Evaluate an AI-generated explanation for clarity, accuracy, and completeness. Practice identifying common problems in AI-generated content.

Time: 30-45 minutes

The Exercise

Step 1: Get an Explanation

Ask an AI tool to explain a technical concept you're somewhat familiar with. Choose something you can verify. Examples:

  • "Explain how HTTPS keeps web traffic secure"
  • "Explain how a recommendation algorithm works"
  • "Explain what happens when you search on Google"
